# 🛡️ Phishing Email Analysis and IOC Extraction Workflow

## 📌 Project Overview

This project demonstrates a structured phishing email investigation workflow based on six real-world phishing scenarios. Each case study was analyzed using a Security Operations Center (SOC) investigation methodology to identify phishing indicators, extract Indicators of Compromise (IOCs), assess potential risks, and document findings in a professional incident report.

The objective of this project is to strengthen practical SOC Level 1 analyst skills by performing phishing investigations and documenting them in a structured, repeatable format.

---

## 🎯 Project Objectives

* Analyze phishing emails using a structured investigation methodology.
* Identify phishing indicators and social engineering techniques.
* Extract Indicators of Compromise (IOCs).
* Perform email content and attachment analysis.
* Assess security risks and potential business impact.
* Identify detection opportunities for security teams.
* Map phishing techniques to the MITRE ATT&CK Framework.
* Recommend mitigation and response actions.

## 🔍 Investigation Methodology

Each phishing email was investigated using the following workflow:

1. Executive Summary
2. Scenario Overview
3. Phishing Techniques Identified
4. Indicators of Compromise (IOCs)
5. Analysis Findings
6. Detection Opportunities
7. Risk Assessment
8. MITRE ATT&CK Mapping
9. Recommended Actions
10. Conclusion
